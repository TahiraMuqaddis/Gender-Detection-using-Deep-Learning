## 🎯 Overview

This project implements a **Gender Detection System** using Deep Learning with TensorFlow/Keras. The system classifies facial images as **Male** or **Female** through a fully connected neural network. A Streamlit web application provides an intuitive interface for real-time predictions.

## ✨ Features

- 🔍 **Real-time Prediction**: Upload images and get instant gender classification
- 📱 **Interactive UI**: Clean and responsive Streamlit interface
- 🧠 **Deep Learning**: 3-layer neural network with dropout regularization
- 📊 **Performance Metrics**: Confidence scores and class probabilities
- 🎨 **Visual Feedback**: Color-coded results and progress bars
- 🔄 **Reproducible**: Fixed random seeds for consistent results
